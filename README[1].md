# IBM HR Analytics: Employee Attrition & Performance:

## 📊 Project Overview:
This project analyzes the IBM HR dataset to understand employee attrition patterns and identify key factors contributing to turnover.

## 👩‍💻 Author:
**Tanvi Sanjay Dighe**  
Data Science Intern 

## 🎯 Objectives:
- Calculate current employee attrition rate
- Analyze demographic factors influencing attrition (age, gender, marital status, education)
- Study job-related factors (job satisfaction, income, overtime, promotion)
- Visualize trends and extract actionable insights

## 🛠️ Tools & Libraries:
- Python, Pandas, NumPy
- Seaborn, Matplotlib
- Jupyter Notebook

## 💡 Key Insights:
- Overall Attrition Rate: ~16.12%
- Higher attrition among young, single employees working overtime
- Job satisfaction, salary hikes, and distance from home significantly impact retention
